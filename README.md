# 📊 Gerador de Relatório CSV

Projeto desenvolvido em Java para registrar produtos e gerar automaticamente um arquivo CSV compatível com o Excel.

## Funcionalidades

* Cadastro do nome do produto
* Registro do preço unitário
* Registro da quantidade
* Cálculo automático do valor total
* Geração de arquivo `.csv`
* Compatibilidade com Microsoft Excel

## Tecnologias

* Java
* PrintWriter
* Arquivos CSV

## Objetivo

Praticar manipulação de arquivos, entrada de dados e cálculos em Java por meio da geração automática de relatórios em formato CSV.
